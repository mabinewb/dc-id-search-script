# 🔍 디시인사이드 식별코드 & IP 검색기 (Deep Scan)

디시인사이드 갤러리 이용 시 고정닉/반고닉의 **식별코드(UID)**나 유동 유저의 **IP**를 기반으로 작성글을 빠르게 추적할 수 있도록 도와주는 유저스크립트입니다.

---

## ✨ 주요 기능

* **강력한 스캔**: 한 번에 30페이지 분량의 게시글을 백그라운드에서 검색합니다.
* **메뉴 통합**: 닉네임을 클릭하면 나타나는 유저 레이어에 [식별코드 검색] 또는 [IP 검색] 버튼이 추가됩니다.
* **본문-목록 연동**: 게시글을 읽는 도중에도 바로 검색을 시작할 수 있으며, 자동으로 목록 페이지로 이동하여 결과를 출력합니다.
* **무한 탐색**: '다음 30페이지 더 검색하기' 버튼을 통해 과거 행적을 끝없이 추적 가능합니다.

1. **Tampermonkey 설치**: 사용하는 브라우저에 맞게 설치하세요.
   - [Chrome용](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) / [Edge용](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpcndfhdbmhlgdicjfibjmocne)
2. **스크립트 설치**: 아래 버튼(혹은 링크)을 클릭하세요.
   - [👉 스크립트 설치하기 (Direct Install)](https://raw.githubusercontent.com/mabinewb/dc-id-search-script/main/dc_search.user.js)
